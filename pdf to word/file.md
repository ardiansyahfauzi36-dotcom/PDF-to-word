# DocConverter - Word to PDF & PDF to Word

Website converter file online gratis.

## Fitur

- ✅ Word (.docx) ke PDF
- ✅ PDF ke Text extraction
- ✅ Drag & drop upload
- ✅ Progress bar
- ✅ Auto delete file setelah 5 menit
- ✅ Mobile responsive
- ✅ Dark modern UI

## Tech Stack

- Node.js
- Express.js
- Multer
- LibreOffice Convert
- PDF Parse
